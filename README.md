# Nani's Knees — Workout App

The interactive workout tracker for subscribers of Nani's Knees Strength Blueprint.

Live at: https://app.naniknees.com

## About

Single-page React app. Tracks workouts, sets, reps, rest timer, weekly progress. All data saved privately to the user's browser via localStorage.

## Running locally

```bash
npm install
npm run dev
```

Opens at http://localhost:5173

## Deployment

Vercel auto-deploys on every push to `main`. Runs `npm run build` and serves `/dist`.

## Stack

- React 18
- Vite
- Lucide React (icons)
